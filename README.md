# Samuel Duval

Personal portfolio built with Astro.

[samuelduval.me](https://samuelduval.me)

## Setup

```bash
npm install
npm run dev
```

## Commands

| Command       | Action                           |
| :------------ | :------------------------------- |
| `npm run dev` | Start dev server at `localhost:4321` |
| `npm run build` | Build production site to `./dist/` |
| `npm run preview` | Preview build locally |
